Kotlin Notebook 是一种交互式工具, 你可以在单个文档内混合代码, 可视化图形, 以及 markdown 文本.
你可以使用 Notebook, 在各个小节中编写和执行代码, 这种小节称为 _代码单元(code cell)_,
你可以立即看到执行结果, 并记录下你的想法.
这样的设置使得 Notebook 称为一种卓越的工具, 适用于快速原型开发, 分析, 和数据科学.
